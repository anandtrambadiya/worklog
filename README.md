# Worklog Dashboard

A simple Worklog Dashboard to manage employees and track time entries.  

---

## Setup & Run

### 1. Backend

Open a terminal:

```bash
git clone https://github.com/anandtrambadiya/worklog.git
cd worklog/backend
python -m venv venv
# Windows
venv\Scripts\activate
# macOS/Linux
# source venv/bin/activate
python app.py
```

Open another terminal

```bash
cd worklog/frontend
npm install
npm start
```
